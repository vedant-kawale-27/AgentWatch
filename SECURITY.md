# Security Policy

## Supported Versions

AgentWatch is under active development. Security fixes are applied to the latest maintained version.

| Version             | Supported |
| ------------------- | --------- |
| Latest main release | ✅         |
| Older releases      | ❌         |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, report them privately by emailing:

**[sreerevanth7b@gmail.com](mailto:sreerevanth7b@gmail.com)**

Please include:

* A description of the vulnerability
* Steps to reproduce
* Potential impact
* Any proof-of-concept if available

You can expect an initial response within 7 days.

If the report is accepted, we will investigate, develop a fix, and coordinate responsible disclosure where appropriate.

We appreciate responsible security research and disclosure.
